Core The Framework
Filosofi: Lightweight, Portable (Shared Hosting Friendly), & High Performance.
Karakteristik: Struktur MVC, Full-CLI (Artisan), Built-in Caching, & Enterprise Patterns (Repository/Service).

🛠️ Fitur & Modul Utama (All-in-One)
Core & Container (app/App/Core):

Container: Implementasi Dependency Injection sederhana.

Config & Lang: Manajemen konfigurasi .env dan multi-bahasa (Localization).

Logging: Sistem pencatatan error/activity ke storage/logs.

HTTP & Routing (app/App/Http):

Router: Mendukung Named Routes, Route Grouping, dan Route Caching untuk speed.

WAF & Middleware: Proteksi bawaan (Web Application Firewall), CSRF, dan Rate Limiting.

Form Request: Validasi input otomatis sebelum masuk ke Controller.

Database & ORM (app/App/Database):

Eloquent-like: Mendukung Model, Relation (1-1, 1-N), dan Query Builder yang chainable.

Migration: Sistem up() dan down() untuk manajemen skema database via CLI.

CLI Artisan Clone (app/Console):

Automation: Puluhan perintah make:* (Controller, Model, Middleware, dll) menggunakan sistem Stubs untuk scaffolding cepat.

Optimization: Perintah optimize dan cache untuk performa di environment terbatas.

Internal System (app/Internal):

Dashboard & Debug: Memiliki view internal untuk monitoring logs, rute, dan status sistem tanpa perlu install tool pihak ketiga.

Error Handling: Custom exception handler untuk tampilan error 404, 500, dan database error yang informatif.

Handlers & Services:

Payment & Mail: Handler siap pakai untuk integrasi gateway pembayaran dan SMTP.

Storage: Manajemen upload (private/public) dengan kontrol akses .htaccess.
