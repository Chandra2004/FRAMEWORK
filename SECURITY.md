# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.0.x   | :white_check_mark: |
| < 5.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within The Framework, please send an e-mail to security@theframework.com. All security vulnerabilities will be promptly addressed.
